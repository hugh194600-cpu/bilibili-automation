# B站弹幕宠物签到挂机小程序 - 规格说明书

## 1. 项目概述

**项目名称**: B站弹幕宠物助手  
**项目类型**: 微信小程序  
**核心功能**: 帮助用户自动化完成B站弹幕宠物的签到、挂机修炼和持续在线监测  
**目标用户**: B站直播观众，弹幕宠物养成用户

## 2. 功能列表

### 2.1 用户登录模块
- B站Cookie登录（手动输入或扫码）
- Cookie有效期检测
- 用户信息展示（UID、昵称、头像）

### 2.2 签到功能
- 每日自动签到
- 签到状态显示（已签到/未签到）
- 签到结果反馈（成功/失败/已签到）

### 2.3 挂机修炼功能
- 选择直播间进行挂机
- 手动开始/停止挂机
- 挂机时长统计
- 模拟弹幕发送（获取经验）

### 2.4 持续在线功能
- 在线状态监测
- 心跳保活机制
- 掉线自动重连
- 在线时长统计

### 2.5 任务中心
- 查看任务状态
- 手动触发各项任务
- 任务执行日志

## 3. 技术方案

### 3.1 前端框架
- 微信小程序原生开发
- 使用TDesign组件库

### 3.2 数据存储
- 微信小程序本地存储（wx.getStorageSync）
- 云开发数据库（可选，用于定时任务）

### 3.3 API调用
- B站直播API
- WebSocket连接（弹幕获取）
- HTTPS请求（签到等操作）

### 3.4 架构设计
```
├── pages/
│   ├── index/          # 首页（状态总览）
│   ├── login/          # 登录页面
│   ├── signin/         # 签到页面
│   ├── hangup/         # 挂机页面
│   └── mine/           # 我的页面
├── utils/
│   ├── api.js          # API封装
│   ├── bilibili.js     # B站相关逻辑
│   ├── websocket.js    # WebSocket管理
│   └── util.js         # 工具函数
├── app.js
├── app.json
└── app.wxss
```

## 4. 页面设计

### 4.1 首页（index）
- 用户信息卡片
- 签到状态卡片
- 挂机状态卡片
- 在线状态卡片
- 快捷操作按钮

- 登录按钮

### 4.3 挂机页面（hangup）
