# B站弹幕宠物助手 - 微信小程序开发完成

## 项目概述
开发了一个用于B站弹幕宠物签到、挂机修炼的微信小程序。

## 完成内容

### 1. 项目结构
```
├── app.js              # 小程序全局逻辑
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── SPEC.md             # 规格说明文档
├── utils/
│   ├── util.js         # 工具函数
│   ├── bilibili.js     # B站API封装
│   └── websocket.js    # WebSocket管理
├── pages/
│   ├── index/          # 首页（状态总览）
│   ├── login/          # 登录页面
│   ├── signin/         # 签到页面
│   ├── hangup/         # 挂机页面
│   └── mine/           # 我的页面
└── assets/icons/        # 图标目录
```

### 2. 核心功能
- **用户登录**: 通过B站Cookie登录
- **每日签到**: 自动签到获取经验
- **挂机修炼**: 选择直播间自动挂机
- **在线监测**: 实时显示挂机状态

### 3. 页面功能
- 首页：状态总览、快捷操作、任务日志
- 登录：Cookie输入、用户协议
- 签到：日历显示、签到记录、奖励说明
- 挂机：直播间选择、实时日志、经验统计
- 我的：用户信息、数据统计、设置

## 使用说明

1. **登录方式**: 需要手动获取B站Cookie后在应用内输入
2. **签到功能**: 每日签到可获得经验值
3. **挂机功能**: 输入直播间ID开始自动挂机，系统会自动发送弹幕获取经验

## 注意事项
- 本工具仅供学习交流使用
- 请妥善保管您的B站Cookie
- 微信小程序无法在后台持续运行，挂机时请保持小程序在前台
