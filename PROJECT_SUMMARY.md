## 📖 使用流程

### 新用户入门
1. 打开小程序
2. 点击"立即登录"
3. 按说明获取B站Cookie
4. 粘贴Cookie并同意协议
5. 点击"立即登录"
6. 进入首页查看状态

### 日常使用
1. **每日签到**：进入签到页 → 点击签到
2. **开始挂机**：进入挂机页 → 输入直播间ID → 开始挂机
3. **查看状态**：在首页查看实时状态
4. **查看统计**：在我的页查看数据统计
