# 微信小程序图标目录

请在此目录下放置以下图标文件：

## TabBar图标（必需）

- `home.png` - 首页未选中状态 (81x81px)
- `home-active.png` - 首页选中状态 (81x81px)
- `signin.png` - 签到未选中状态 (81x81px)
- `signin-active.png` - 签到选中状态 (81x81px)
- `hangup.png` - 挂机未选中状态 (81x81px)
- `hangup-active.png` - 挂机选中状态 (81x81px)
- `mine.png` - 我的未选中状态 (81x81px)
- `mine-active.png` - 我的选中状态 (81x81px)

## 图标要求
- 尺寸：81x81像素（建议）
- 格式：PNG
- 背景透明

## 提示
如果没有自定义图标，微信小程序会使用占位图标。
